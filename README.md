# Website Clone - Automated Deployment

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](https://https://github.com/Melkor-1/task-1/edit/main/LICENSE)

This repository contains a template of the **Internee.pk** website. It is designed to be deployed on an **Amazon EC2 instance**. The repository includes a **Bash script** that automates the process of setting up a server environment, installing required packages, and hosting the website with Apache HTTP Server.

## Usage

1. Clone the repository:

```bash
git clone --depth 1 https://github.com/Melkor-1/task-1.git
cd task-1
```

2. Make the script executable:

```bash
chmod +x deploy.sh
```

3. Run the script:

```bash
./deploy.sh
```

### Note:
The script is general and can be edited and used as a general deployment script.
